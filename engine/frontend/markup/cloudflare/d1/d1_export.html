Export database

Limitations

  1. Export is not supported for virtual tables, including databases with virtual tables. As a workaround, delete any virtual tables, export, and then recreate virtual tables.
  2. A running export will block other database requests.
  3. Any numeric value in a column is affected by JavaScript's 52-bit precision for numbers. If you store a very large number (in int64), then retrieve the same value, the returned value may be less precise than your original number.
